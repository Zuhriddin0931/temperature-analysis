# Temperature Analysis in C

## 🇺🇸 English

### Description

This project is a simple C program that stores temperature data for seven days and performs basic analysis.

### Features

* Reads daily temperature records
* Finds the hottest day
* Finds the coldest day
* Calculates the average temperature

### Technologies

* C Programming Language
* Structures
* Functions
* Arrays
* Pointers

### Example Input

```text
1 25
2 27
3 21
4 30
5 28
6 19
7 24
```

### Example Output

```text
Hottest day: 4 30
Coldest day: 6 19
Average: 24.86
```

Created as a learning project for practicing structures, arrays, functions, and pointers in C.
